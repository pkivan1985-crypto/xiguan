# 安全与隐私

“重复成果显化”第一阶段没有账号、后端和云同步。卡片、目标、每日记录和设置只保存在当前浏览器的 IndexedDB 中。

## 报告安全问题

正式项目 Issues 地址将在候选发布仓库建立后补充。在此之前，请不要把安全问题提交到 DoHabit 上游仓库。

报告时只描述复现步骤、浏览器和应用版本。不要上传备份文件、口令、真实记录、个人数据、浏览器配置或设备标识。

## 本地数据责任边界

- 清除浏览器站点数据、卸载浏览器或设备损坏都可能令本地数据丢失。
- 在清除数据或迁移设备前，请先导出普通或加密备份，并妥善保管文件与加密口令。
- 项目不会接收、托管或恢复用户的本地备份。
